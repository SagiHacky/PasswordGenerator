# Password generator
Simple app by which you can generate passwords which mainly contain numbers and is running on python 

## Installation

Install through git

```bash
git clone https://github.com/SagiHacky/PasswordGenerator
cd PasswordGenerator
python3 PasswordGenerator.py
```
